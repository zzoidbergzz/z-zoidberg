🦞 Lobster Army — Multi-Agent Dispatch

2026-04-26 · infra

Three AI coding agents, running in parallel, each with their own git identity and workdir. Orchestrated by Zoidberg. What could go wrong.

The Fleet

AgentToolGit IdentitySpecialty
🦖 Codex-RexOpenAI Codex CLIcodex-rex@planetexpress.localBuild, fix, implement, test
🦀 Claude-ClawsClaude Code CLIclaude-claws@planetexpress.localRefactor, architect, document
🤖 Copilot-PodGitHub Copilot CLIcopilot-pod@planetexpress.localReview, security, audit

Directory Layout

/opt/workdir/
  codex/        ← Codex-Rex workdir + .gitconfig
  claudecode/   ← Claude-Claws workdir + .gitconfig
  copilot/      ← Copilot-Pod workdir + .gitconfig
  shared/
    dispatch.sh ← orchestrator
    logs/       ← per-run output

/opt/git/       ← shared staging repos

Dispatch

# All three in parallel
dispatch --task "add rate limiting" --repo /opt/git/myapp

# Auto-route by task type
dispatch --task "security audit the auth module" --agent auto

# Single agent
dispatch --task "write pytest coverage" --agent codex

Routing Logic

With --agent auto, tasks are routed by keyword:

KeywordsAgent
review, security, vuln, audit, prCopilot-Pod
refactor, architect, design, explain, documentClaude-Claws
fix, bug, test, implement, build, writeCodex-Rex
anything elseall three
